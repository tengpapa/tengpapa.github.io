烧鲅鱼做法

发表于 2017-03-30 | 分类于 美食

一. 配料

鲅鱼（切块）、花椒、八角、葱、姜、蒜、油、盐、酱油、醋、料酒、糖、蚝油。

二. 流程

  1. 锅内多放油，小火，放入花椒和八角（掰开）炸香。

  2. 放入葱、姜、蒜，慢炒，可以放入少量的五花肉片。

  3. 加入酱油、料酒、醋（醋要多放，4-5铲），有料酒可以不用加白酒，或者放点泡酒的枸杞等。

  4. 烧热后，加水，盖锅盖，将水烧开。

  5. 放入鲅鱼，鱼不新鲜可加蚝油，小火开炖，炖到一半，可尝味酌情加盐和糖。

  6. 一直小火炖，收汤，出锅。

三. 注意

  1. 多放油和醋

  2. 千炖豆腐，万炖鱼
