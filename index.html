重装系统后，重新配置GitHub上的博客

发表于 2018-04-01 | 分类于 博客

重装系统以后，发现’hexo deploy’无法连上GitHub了。查了一下，原来本地存放的GitHub上配置的SSH的公钥和私钥已经不见了。下面就重新配置吧：

  1. hexo之前的配置在此就不讲了。
  2. 打开git bash，运行 ‘ssh-keygen -t rsa -C “youremail@example.com”‘ 。运行完以后，会要求你输入密码，可输可不输，如果输入，hexo deploy的时候会要求你输入该密码，该密码的作用应该是将私钥加密进行本地存放。默认的目录下会生成.ssh目录，里面有id_rsa和id_rsa.pub两个文件。
  3. 登陆GitHub，打开「Settings」->「SSH and GPG keys」，然后点击「new SSH key」，填上任意Title，在Key文本框里粘贴公钥id_rsa.pub文件的内容，最后点击「Add SSH Key」，就可以看到已经添加的Key。至此，在GitHub网站上的设置已经完成了。
  4. 运行’git config –global user.email “you@example.com”‘，’git config –global user.name “Your Name”‘。
  5. 然后就可以发布网站了。

烧鲅鱼做法

发表于 2017-03-30 | 分类于 美食

一. 配料

鲅鱼（切块）、花椒、八角、葱、姜、蒜、油、盐、酱油、醋、料酒、糖、蚝油。

二. 流程

  1. 锅内多放油，小火，放入花椒和八角（掰开）炸香。

  2. 放入葱、姜、蒜，慢炒，可以放入少量的五花肉片。

  3. 加入酱油、料酒、醋（醋要多放，4-5铲），有料酒可以不用加白酒，或者放点泡酒的枸杞等。

  4. 烧热后，加水，盖锅盖，将水烧开。

  5. 放入鲅鱼，鱼不新鲜可加蚝油，小火开炖，炖到一半，可尝味酌情加盐和糖。

  6. 一直小火炖，收汤，出锅。

三. 注意

  1. 多放油和醋

  2. 千炖豆腐，万炖鱼
